# PanoramaBetter
This is something I made in an hour to prove that we can use html elements as hotspot indicators in a 3d panorama view.

Groetjes Folkert <3

## Installing
```
git clone https://github.com/folkerthoogenraad/PanoramaBetter.git
cd PanoramaBetter
npm install
```

## Running
```
npm run dev
```

## Building
```
npm run build
```
